Irid.JS Intro

Irid.JS is a simple JavaScript framework by Aeonic. It helps shorten your code as elements appear and disappear, and makes it look cleaner and easier to read. In addition, it is perfect for beginners because it uses simple language that we naturally use in human speech. You can also set program defaults, make random numbers easier, and much much more!

Getting access to all of the elements in Irid.JS

Irid.JS is not included with HTML or JavaScript. You put in the following code in the bottom of the <body> tag of your HTML.

<script src="https://www.aeonic.eu.org/iridium/irid.js"></script>

Reference

eShow()

The function eShow() displays an element.

Code

<button class="e-transparent" onclick="eShow('ex1btn');">Click me!</button><br/><button class="e-hide e-transparent" id='ex1btn'>I just appeared!</button>

Result


eHide()

The function eHide() hides an element.

Code

<button class="e-transparent" onclick="eHide('ex2btn');">Click me!</button><br/><button class="e-transparent" id='ex2btn'>Click above to make me disappear!</button>

Result
