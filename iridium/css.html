Irid.CSS Intro

Irid.CSS is a CSS framework that allows anybody to easily style HTML documents without creating a lot of CSS. HTML Buttons can be converted. You can put an entire page in Dark Mode.

Irid.CSS also shortens your code. Rather than typing style="display: none;" to hide an element by default, you can put class="e-hide".

Getting access to all of the elements in Irid.CSS

Irid.CSS is not included with HTML. You put in the following code in the <head> tag of your HTML.

<link href="https://www.aeonic.eu.org/iridium/irid.css" rel="stylesheet" type="text/css">.

Alternatively, you can use @import in standard CSS to include it as well.

@import url("https://aeonic.ebp11.repl.co/iridium/irid.css/").

Hide elements

With Irid.CSS, you can easily add one class to hide an HTML element. Put in the e-hide class into any HTML element. We made a blue, 15 pixel square with a div.

HTML Code

<div class="e-hide" style="height: 15px; width: 15px; background-color: blue;"></div>

Result

Show elements

In the previous example, we were able to hide a <div> element. But how do we show it?

Irid.CSS has a class for it!

The e-block class shows the otherwise hidden element. Now for the <div>.

HTML Code

<div class="e-block" style="height: 15px; width: 15px; background-color: blue;"></div>

Result

Explanation

The div showed since we put a class that prevented it from disappearing.

Transparent elements

With the e-transparent class, you can make an element disappear, but the text stay. Let us make a button with the text Click me. We will use the class to make you only see the text. If you hover over the button, you can see a border to see where the entire element is. You can use the e-transparent-no-border class to remove the border on hover. If you really want to keep the element secret, use the e-transparent-full class. (That is one easy way to keep all your secrets!) (For e-transparent-full, you may also use e-secret. Both have no support for images.)

HTML Code

<button class="e-transparent">Click me</button>

Result

HTML Code

<button class="e-transparent-no-border">Click me</button>

Result

HTML Code

<button class="e-transparent-full">Click me</button>

Result

Dark Mode on a <div>

It is nighttime, but it feels strange. The webpage is showing up too white. You want to add a dark mode to your website. With Irid.CSS, that becomes easy, since there is a class for that. The e-dark-mode class turns the background black, and the text white. All images come in greyscale.

Draw some simple shapes

Irid.CSS has some classes you can put on <div> and <span> elements to display shapes. The shapes are 20px and black by default, so to customise them, you must add CSS.

Circles/Ovals

You can use the e-circle class to create a circle or oval.

Code: <span class="e-circle"></span>
Result: cir

Squares/Rectangles

You can use the e-rect class to create a square or rectangle.

Code: <span class="e-rect"></span>
Result: sq

Triangle

You can use the e-triangle class to create a triangle. (Note that it supports only <div> elements.)

Code: <div class="e-triangle"></div>
