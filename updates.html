    How Build Numbers Work

    Eyynxtxzz.w

    • Eyy ...: last 2 digits of the year
    • x ... x: the version number
    • ... .w: the build of the version
    • ... zz or ... zz. ...: the the part of the version:
      • Ax: Alpha #x
      • Bx: Beta #x
      • Gx: Gamma #x
      • AR: Public Alpha Release
      • BR: Public Beta Release
      • GR: Public Gamma Release
      • PR: Final Public Release
      • RR: Revised Public Release
      • IS: Induvidually Customised Release
